#### 🎓ទិដ្ឋភាពទូទៅ

នេះគឺជាទិដ្ឋភាពទូទៅដែលយើងនឹងត្រូវរៀនអំពីមុខងាររបស់ NextJS ជាមួយគ្នាដូចខាងក្រោម៖

- **Styling**: ដឹងពីរបៀបមួយចំនួនក្នុងការប្រើប្រាស់ Style នៅក្នុង NextJS application
- **Optimizations**: ធ្វើយ៉ាងដូចម្តេចដើម្បីបង្កើននូប្រសិទ្ធភាពរបស់ Images, Link, និង Fonts
- **Routing**: ធ្វើយ៉ាងដូចម្តេចដើម្បីរៀបចំ Database នៅលើ Vercel និងការអនុវត្តក្នុងការទាញទិន្នន័យ និងការ Streaming
- **Search និង Pagination**: ធ្វើយ៉ាងដូចម្តេចដើម្បីបង្កើតមុខងារក្នុង Search និង Pagination ដោយប្រើប្រាស់នូវ URL Search Params
- **Mutating Data**: ធ្វើដូចម្តេចដើម្បី mutate data ដោយប្រើប្រាស់នូវ React Server Component និង​ Revalidate NextJS Cache
- **Error handling**: ធ្វើដូចម្តេចដើម្បីគ្របគ្រង់ error page ទូទៅ និង404 Not Found error
- **Accessibility**: ធ្វើដូចម្តេចដើម្បីធ្វើ form validation នៅ server-side និង គន្លឹះក្នុងការ improve accessibility
- **Authentication**: ធ្វើដូចម្តេចដើម្បីរៀបចំ Authentication ដោយប្រើប្រាស់នូវ NextAuth.js និង Middleware
- **Metadata**: ធ្វើដូចម្តេចដើម្បីរៀបចំការចែករំលែកទៅ Social Media
